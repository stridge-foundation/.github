[Stridge](https://stridge.com) is crypto payment infrastructure for fintechs. One gateway accepts deposits on any supported chain, picks the cheapest route automatically, and settles to whichever destination chain and token you choose — and the same gateway powers withdrawals. You integrate once, against one contract, and your users never have to bridge, swap, or pick a network again.

### Build with Stridge

- **[Gateway Kit](https://docs.stridge.com/gateway)** — drop-in React components for deposits and withdrawals.
- **[TypeScript SDK](https://docs.stridge.com/sdk)** — a typed, dependency-free client for any backend or app.
- **[Universal Deposit Addresses](https://docs.stridge.com/uda)** — the settlement model that powers every flow.
- **[Supported networks](https://docs.stridge.com/networks)** — every chain and token Stridge routes through.

To learn more, check out our [docs](https://docs.stridge.com).
